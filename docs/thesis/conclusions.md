# Загальні висновки

У дипломній роботі вирішено актуальне науково-практичне завдання з розробки комп'ютерно-інтегрованої системи короткострокового прогнозування сонячної генерації ФЕС на базі гібридних нейронних мереж CNN-LSTM. Впровадження системи дозволяє істотно підвищити точність прогнозування та зменшити фінансові втрати енергогенеруючих компаній від небалансів в умовах сучасного балансуючого ринку електричної енергії України.

Основні наукові та практичні результати роботи полягають у наступному:

1. **Досліджено погодні чинники впливу на сонячну генерацію:** Проведено кореляційний аналіз між метеопараметрами та вихідною потужністю ФЕС. Встановлено, що глобальне горизонтальне випромінювання (GHI) має найсильніший позитивний зв'язок із генерацією (коефіцієнт кореляції Пірсона $r = 0.94$), тоді як температура повітря та відносна вологість чинять непрямий модулюючий вплив на КВВП сонячних панелей.
2. **Розроблено асинхронний конвеєр передобробки даних:** Створено програмні модулі для імпорту історичної телеметрії ФЕС потужністю 30 кВт та відповідних погодних даних Open-Meteo API. Реалізовано методи об'єднання даних, заповнення пропусків, нормалізації та формування ковзних вікон.
3. **Проектовано та реалізовано гібридну архітектуру CNN-LSTM:** Поєднання одновимірних згорткових шарів (Conv1D) для вилучення просторових ознак та осередків довгої короткострокової пам'яті (LSTM) для аналізу часових послідовностей забезпечило високу якість навчання. Базова модель досягла стабільної точності з похибкою $MAPE = 9.06\%$ та $MAE = 0.2711$ кВт на тестовій вибірці.
4. **Розроблено інфраструктуру бази даних та API:** Для оптимізації збереження часових рядів розгорнуто СУБД PostgreSQL з розширенням TimescaleDB. Спроектовано та реалізовано REST API на базі FastAPI із використанням асинхронних патернів роботи з базою даних, що забезпечує високу швидкість обробки запитів прогнозування та телеметрії.
5. **Впроваджено та досліджено метод онлайн-навчання:** Запропоновано нову концепцію мультимасштабного темпорального ансамблю моделей (Base + Year + Month), яка оновлює ваги нейромережі на ковзному вікні свіжої телеметрії. Експериментально доведено перевагу ансамблювання, що знизило $MAPE$ до **$8.54\%$** (покращення на $5.77\%$ відносно базової моделі), повністю розв'язавши проблему кумулятивного дрейфу ваг та перенавчання (забування).
6. **Виконано техніко-економічне обґрунтування системи:** 
   * Розраховано нормовану вартість електроенергії (LCOE): для сонячної ФЕС собівартість становить **3.63 грн/кВт-год**, що є дешевше за вугільну генерацію (4.24 грн/кВт-год) та газову пікову (4.96 грн/кВт-год).
   * Проведено розрахунки фінансових втрат від небалансів на балансуючому ринку України (ціна 4.5 грн/кВт-год). Встановлено, що розроблена система прогнозування знижує витрати на сплату штрафів за дисбаланси на **81.93%** порівняно з наївними методами без ML.
   * Річний економічний ефект (заощадження) від впровадження системи для промислової ФЕС потужністю 10 МВт становить **16 147 746 грн**, що підтверджує надзвичайно високу комерційну привабливість проекту.

Розроблене програмне та інформаційне забезпечення має модульну структуру технічних шарів (Technical Layers Architecture), є потокобезпечним та повністю готове для впровадження на реальних сонячних електростанціях з метою інтеграції зі SCADA-системами та автоматизації подачі добових графіків генерації.
