# Вступ

## Актуальність теми

Сучасний етап розвитку світової енергетики характеризується стрімким збільшенням частки відновлюваних джерел енергії (ВДЕ), серед яких сонячні фотоелектричні станції (ФЕС) посідають одне з провідних місць. Відповідно до енергетичної стратегії України, збільшення генерації з чистих джерел є стратегічним пріоритетом для забезпечення енергетичної незалежності, диверсифікації паливно-енергетичних ресурсів та зниження антропогенного навантаження на довкілля.

Проте інтеграція великої кількості сонячних станцій в Об'єднану енергосистему (ОЕС) України супроводжується значними технологічними викликами. Головним з них є високий рівень стохастичності та нестабільності сонячної генерації, що безпосередньо залежить від швидкозмінних метеорологічних факторів (інтенсивності сонячного випромінювання, хмарності, температури тощо). Непрогнозовані коливання потужності ФЕС вимагають додаткових резервів маневреної генерації (переважно вугільної та газової) для балансування системи, що знижує загальну екологічну та економічну ефективність використання сонячної енергії.

З впровадженням нової моделі ринку електричної енергії в Україні виробники з ВДЕ несуть повну фінансову відповідальність за спричинені дисбаланси — розбіжності між заявленими добовими графіками відпуску та фактичною генерацією. Штрафи на балансуючому ринку суттєво знижують рентабельність сонячних електростанцій. У зв'язку з цим, розробка та впровадження комп'ютерно-інтегрованих систем короткострокового прогнозування генерації ФЕС з високою точністю є надзвичайно актуальним науково-технічним завданням.

Традиційні статистичні методи (наприклад, ARIMA) та класичні моделі машинного навчання (SVM, Random Forest) не завжди здатні ефективно моделювати складні нелінійні просторово-часові залежності у погодних даних та телеметрії ФЕС. Перехід до технологій глибокого навчання (Deep Learning), зокрема до гібридних архітектур, що поєднують згорткові (CNN) та рекурентні (LSTM) нейронні мережі, відкриває нові можливості для суттєвого підвищення точності прогнозування.

## Мета і завдання дослідження

**Метою роботи** є розробка комп'ютерно-інтегрованої системи короткострокового прогнозування генерації фотоелектричних станцій на основі гібридних нейронних мереж архітектури CNN-LSTM для зниження фінансових втрат від небалансів та підвищення стабільності функціонування енергосистеми.

Для досягнення поставленої мети було визначено та вирішено такі **завдання**:
1. Провести аналіз сучасного стану систем прогнозування генерації ФЕС, вивчити вплив метеорологічних та географічних чинників на сонячну генерацію.
2. Розробити програмні компоненти для збору, очищення та передобробки первинних історичних телеметричних та метеорологічних даних.
3. Проектувати та реалізувати архітектуру гібридної нейронної мережі CNN-LSTM для вилучення просторово-часових ознак з часових рядів.
4. Розробити інфраструктуру бази даних часових рядів на основі PostgreSQL/TimescaleDB та асинхронний REST API на FastAPI для надання інтерактивного інтерфейсу прогнозування споживачам.
5. Провести експериментальні дослідження точності розробленої моделі, виконати порівняльний аналіз із базовими методами.
6. Розробити та дослідити стратегію онлайн-навчання та темпорального ансамблювання моделей для адаптації до динамічних змін погоди в реальному часі.
7. Виконати техніко-економічне обґрунтування розробленої системи, розрахувати LCOE та фінансовий ефект від зниження штрафів за дисбаланси.

## Об'єкт, предмет та методи дослідження

**Об'єктом дослідження** є процес короткострокового прогнозування генерації електроенергії на фотоелектричних станціях в умовах динамічних метеорологічних змін.

**Предметом дослідження** є математичні моделі гібридної архітектури CNN-LSTM, алгоритми передобробки та масштабування часових рядів, а також структура програмного забезпечення для побудови асинхронних комп'ютерно-інтегрованих систем прогнозування.

**Методи дослідження** базуються на теорії штучних нейронних мереж, глибокого навчання (шари одновимірної згортки Conv1D для просторового аналізу та осередки LSTM для моделювання часових залежностей), методах математичної статистики (кореляційний аналіз Пірсона та Спірмена), теорії баз даних та асинхронного веб-програмування.

## Наукова новизна отриманих результатів

Наукова новизна роботи полягає у **модифікації стратегії онлайн-адаптації прогнозування сонячної генерації шляхом впровадження мультимасштабного темпорального ансамблю моделей (Base + Year + Month)**. На відміну від стандартних підходів кумулятивного fine-tuning, які призводять до дрейфу ваг нейромережі та локального перенавчання (забування глобальних сезонних патернів), запропонована стратегія поєднує базову модель (навчену на довгостроковій ретроспективі) та два незалежні клони, донавчені на річному та місячному горизонтах. Це дозволяє одночасно враховувати як сезонні кліматичні коливання, так і поточні локальні зміни погодних умов, що забезпечує зниження середньої абсолютної похибки MAPE на $5.77\%$ відносно статичної моделі.

## Практичне значення отриманих результатів

Практичне значення розробки полягає у створенні готового програмного комплексу комп'ютерно-інтегрованої системи прогнозування, який містить:
1. Модуль збору даних телеметрії ФЕС та метеопрогнозів з Open-Meteo.
2. Масштабовану базу даних на базі PostgreSQL + TimescaleDB, яка оптимізована для високошвидкісного запису часових рядів.
3. Асинхронний REST API інтерфейс на FastAPI з автоматичною документацією Swagger UI, що спрощує інтеграцію системи прогнозування із зовнішніми промисловими системами керування (SCADA ФЕС, диспетчерські центри оператора системи розподілу).
4. Реєстр моделей `ModelRegistry` з механізмом потокобезпечного конкурентного доступу до ваг моделей під час інференсу та онлайн-навчання.
5. Розроблені інструменти безпосередньо трансформуються у зниження фінансових втрат від небалансів ФЕС (заощадження понад 16 млн грн/рік для промислової ФЕС потужністю 10 МВт).
