# AGENTS.md

> Цей файл призначений для ШІ-агентів і містить опис структури проекту та базові правила. Оновлюйте його у разі суттєвих змін архітектури.

## Огляд проекту
Комп'ютерно-інтегрована система короткострокового прогнозування генерації фотоелектричних станцій (ФЕС) на основі гібридних нейронних мереж CNN-LSTM.

## Технологічний стек
- **Мова програмування:** Python
- **Фреймворк:** FastAPI + TensorFlow/Keras
- **База даних:** PostgreSQL (з розширенням TimescaleDB)
- **ORM:** SQLAlchemy (з Alembic для міграцій)

## Структура проекту
```
.
├── .agent/                 # Внутрішня конфігурація, скрипти та воркфлоу ШІ-агентів
├── .agents/                # Встановлені зовнішні та академічні навички агентів
├── .ai-factory/            # Конфігурації та опис життєвого циклу проекту
│   ├── rules/              # Правила та стандарти кодування для конкретних зон
│   │   └── base.md         # Базові правила та конвенції проекту
│   ├── DESCRIPTION.md      # Загальний технічний опис проекту та стеку
│   └── config.yaml         # Конфігурація поведінки та шляхів AI Factory
└── .ai-factory.json        # Файл стану встановлених плагінів та навичок
```

## Ключові точки входу
| Файл | Призначення |
|------|-------------|
| `.ai-factory/config.yaml` | Основний файл конфігурації роботи системи AI Factory |
| `.ai-factory/rules/base.md` | Базові правила оформлення коду, логування та обробки помилок |

## Правила для ШІ-агентів
- Не об'єднуйте Git-команди в один рядок, якщо вони залежать від стану репозиторію. Розділяйте їх для безпечного виконання.
