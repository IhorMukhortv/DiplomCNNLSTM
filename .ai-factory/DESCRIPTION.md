# Комп'ютерно-інтегрована система короткострокового прогнозування генерації ФЕС на основі CNN-LSTM

## Огляд
Комп'ютерно-інтегрована система короткострокового прогнозування генерації фотоелектричних станцій (ФЕС) призначена для точного прогнозування обсягів виробництва електроенергії на основі історичних даних генерації, поточних та прогнозних метеорологічних факторів (інтенсивність сонячного випромінювання, температура, хмарність тощо).

Система базується на гібридній нейронній мережі CNN-LSTM (Convolutional Neural Network - Long Short-Term Memory). Шар CNN відповідає за вилучення просторових та локальних характеристик із вхідних часових рядів, тоді як LSTM фокусується на моделюванні довгострокових та короткострокових часових залежностей у часових послідовностях.

## Основні функції
- **Збір та передобробка даних**: Імпорт історичних даних генерації ФЕС, метеорологічних параметрів та прогнозів погоди; очищення даних, заповнення пропусків, масштабування та нормалізація.
- **Гібридне прогнозування CNN-LSTM**: Навчання та запуск нейромережевої моделі для прогнозування генерації на короткостроковий період (від 15 хвилин до 24-48 годин вперед).
- **REST API інтеграція**: API для надання результатів прогнозування зовнішнім системам (SCADA, системи керування енергомережами, диспетчерські пункти).
- **Керування моделями**: Збереження, версіонування та автоматичне донавчання моделей.
- **Візуалізація та моніторинг**: Оцінка метрик точності (MAE, RMSE, MAPE) та порівняння фактичної і прогнозованої генерації.

## Технологічний стек
- **Мова програмування:** Python
- **Фреймворк та ML-ядро:** FastAPI, TensorFlow / Keras (для CNN-LSTM)
- **База даних:** PostgreSQL (з розширенням TimescaleDB для оптимізації роботи з часовими рядами)
- **ORM:** SQLAlchemy (з Alembic для міграцій)
- **Інструменти аналізу даних:** Pandas, NumPy, Scikit-learn
